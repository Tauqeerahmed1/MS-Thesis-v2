Database-ready schema and data structure for storing and managing your Press Release Categories within the Evertise AI PR system.
This structure is designed for:
* AI prompt generation (category-based writing styles),
* Press release classification, and
* Smart distribution targeting (by category or industry).

?? 1. Database Schema: press_release_categories
CREATE TABLE `press_release_categories` (
  `id` INT AUTO_INCREMENT PRIMARY KEY,
  `category_name` VARCHAR(255) NOT NULL,
  `slug` VARCHAR(255) NOT NULL UNIQUE,
  `parent_id` INT DEFAULT NULL,
  `description` TEXT DEFAULT NULL,
  `created_at` TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  FOREIGN KEY (`parent_id`) REFERENCES `press_release_categories`(`id`) ON DELETE SET NULL
);

Workflow Summary
1. User fills the PR form ? submits metadata (title, topic, category, keywords, target market, tone).
2. System saves the PR into press_releases table.
3. Distribution sites are loaded from distribution_sites.
4. Numeric scoring: fast local calculation of relevance.
5. GPT semantic scoring: AI evaluates contextual fit.
6. Hybrid final score: weighted sum of numeric + GPT scores.
7. Save matches in distribution_matches for history and analytics.
8. Top recommendations displayed to the user (packages & premium websites).
